# API Documentation

## Authentication
- POST /api/v1/auth/register
- POST /api/v1/auth/login
- POST /api/v1/auth/refresh

## Incidents
- GET /api/v1/incidents
- POST /api/v1/incidents

- PUT /api/v1/incidents/{id}
- DELETE /api/v1/incidents/{id}

## Logs

- POST /api/v1/logs

## Dashboards

- POST /api/v1/dashboards/widgets

## Alerts
